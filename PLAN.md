# PLAN.md — Leave management module

_Generated by Gestalt planning layer (feature 8908339f)._

## Feature description

Build the leave management module. Employees can apply for annual, sick, and emergency leave. Managers approve or reject requests. The system tracks leave balances and prevents overlapping requests.

## Modules
- **leave** (`src/modules/leave`) — owns: leave.model.ts, leave.repository.ts, leave.service.ts, leave.routes.ts, leave.test.ts
- **balance** (`src/modules/balance`) — owns: balance.model.ts, balance.repository.ts, balance.service.ts
- **policy** (`src/modules/policy`) — owns: policy.model.ts, policy.repository.ts, policy.service.ts, policy.routes.ts
- **employee** (`src/modules/employee`) — owns: employee.model.ts, employee.repository.ts, employee.service.ts, employee.routes.ts

## Domain entities
- **LeaveRequest** — Tracks leave requests made by employees and their statuses. (attributes: id, employeeId, type, startDate, endDate, status, managerId, managerComment, createdAt)
- **LeaveBalance** — Manages the leave balances for each employee. (attributes: employeeId, leaveType, totalDays, usedDays, year)
- **LeavePolicy** — Defines the rules and limits for different types of leave. (attributes: id, leaveType, defaultDaysPerYear, maxConsecutiveDays, requiresApproval, createdAt)

# PLAN.md — Build the leave management module

_Generated by Gestalt planning layer (feature ec42e085)._

## Feature description

Build the leave management module. Employees apply for annual, sick, and emergency leave. Managers approve or reject. System tracks leave balances.

## Modules
- **leave** (`src/modules/leave`) — owns: LeaveRequest, leave application workflow, leave request submission, approval and rejection state transitions
- **balance** (`src/modules/balance`) — owns: LeaveBalance, leave balance initialization, leave balance adjustments
- **employee** (`src/modules/employee`) — owns: Employee, manager relationship data
- **policy** (`src/modules/policy`) — owns: LeavePolicy, leave entitlement rules
- **notification** (`src/modules/notification`) — owns: Notification, leave workflow notifications

## Domain entities
- **LeaveRequest** — Represents a leave application and its approval lifecycle. (attributes: id, employeeId, leaveType, startDate, endDate, status)
- **LeaveBalance** — Tracks remaining leave entitlement for an employee. (attributes: employeeId, leaveType, availableDays)
- **Employee** — Provides employee identity and reporting relationships. (attributes: id, managerId, name)
- **LeavePolicy** — Defines leave eligibility and entitlement rules. (attributes: id, leaveType, annualEntitlement)
- **Notification** — Represents workflow notifications generated by leave events. (attributes: id, recipientEmployeeId, type)

# PLAN.md — Build the leave management module

_Generated by Gestalt planning layer (feature 7ab81ea3)._

## Feature description

Build the leave management module. Employees apply for annual, sick, and emergency leave. Managers approve or reject. System tracks leave balances.

## Modules
- **Leave** (`src/modules/leave`) — owns: Leave request domain model, Leave request persistence, Leave request business workflow, Leave request HTTP routes, Approval and rejection orchestration, Overlap validation
- **Balance** (`src/modules/balance`) — owns: Leave balance domain model, Leave balance persistence, Balance availability checks, Balance usage recording
- **Employee** (`src/modules/employee`) — owns: Employee domain model, Employee persistence, Employee lookup, Manager-subordinate relationship validation
- **Policy** (`src/modules/policy`) — owns: Leave policy domain model, Leave policy persistence, Policy lookup for leave validation
- **Shared Database** (`src/shared/db`) — owns: PostgreSQL connection pool, Shared database access foundation
- **Shared Middleware** (`src/shared/middleware`) — owns: Authentication enforcement, Role-based route access, Centralized error handling
- **Shared Types** (`src/shared/types`) — owns: Shared enums, Shared error classes, Cross-module primitive types

## Domain entities
- **Employee** — Represents the person applying for leave, approving subordinate leave, or administering leave data. (attributes: id, name, email, role, managerId)
- **LeaveRequest** — Represents an employee leave application and its approval lifecycle. (attributes: id, employeeId, type, startDate, endDate, status, managerId, managerComment, createdAt)
- **LeaveBalance** — Tracks available and consumed leave entitlement for an employee by leave type and year. (attributes: employeeId, leaveType, totalDays, usedDays, year)
- **LeavePolicy** — Defines configurable rules for each leave type used when validating requests and initializing balances. (attributes: id, leaveType, defaultDaysPerYear, maxConsecutiveDays, requiresApproval, createdAt)

## Feature description

Build the leave management module. Employees apply for annual, sick, and emergency leave. Managers approve or reject. System tracks leave balances.

## Domain entities
- **LeaveRequest** — Tracks leave requests submitted by employees for various leave types. (attributes: id, employeeId, type, startDate, endDate, status, managerId, managerComment, createdAt)
- **LeaveBalance** — Maintains the leave balance for each employee based on leave type. (attributes: employeeId, leaveType, totalDays, usedDays, year)
- **LeavePolicy** — Defines the rules and limits for different types of leave. (attributes: id, leaveType, defaultDaysPerYear, maxConsecutiveDays, requiresApproval, createdAt)

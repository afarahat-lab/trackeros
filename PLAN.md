# PLAN.md — Build the leave management module for the corporate operations system

_Generated by Gestalt planning layer (feature f070332a)._

## Feature description

Build the leave management module for the corporate operations system. Employees can submit leave requests with leave type, start date, end date, and a justification note. Managers approve, reject, or request changes. The system tracks a remaining balance per leave type per employee and decrements on approval. All approval and rejection actions write an audit record with the actor, timestamp, and reason. Include API routes, services, repositories, domain models, and unit tests. Use the project's declared stack.

## Modules
- **leave** (`src/modules/leave`) — owns: LeaveRequest, LeaveAuditRecord, leave request submission workflow, leave request state transitions, manager approval actions
- **balance** (`src/modules/balance`) — owns: LeaveBalance, leave balance adjustments
- **employee** (`src/modules/employee`) — owns: Employee, manager relationship data, role information
- **policy** (`src/modules/policy`) — owns: LeavePolicy, leave entitlement rules
- **notification** (`src/modules/notification`) — owns: Notification, leave workflow notifications

## Domain entities
- **LeaveRequest** — Represents a leave application and its workflow state. (attributes: id, employeeId, leaveType, startDate, endDate, justification, status)
- **LeaveBalance** — Tracks remaining leave entitlement for an employee by leave type. (attributes: employeeId, leaveType, remainingDays)
- **Employee** — Represents employees and manager reporting relationships used in leave approval workflows. (attributes: id, managerId, role)
- **LeavePolicy** — Defines leave entitlement and leave-type rules. (attributes: id, leaveType, entitlementDays)
- **Notification** — Represents workflow notifications related to leave requests. (attributes: id, recipientEmployeeId, message, status)
- **LeaveAuditRecord** — Captures approval workflow audit history for all state-changing leave actions. (attributes: id, leaveRequestId, actorEmployeeId, action, reason, timestamp)

## Feature description

Build the leave management module. Employees apply for annual, sick, and emergency leave. Managers approve or reject. System tracks leave balances.

- **employee** (`src/modules/employee`) — owns: employee.model.ts, employee.repository.ts, employee.service.ts, employee.controller.ts, employee.routes.ts
- **balance** (`src/modules/balance`) — owns: balance.model.ts, balance.repository.ts, balance.service.ts, balance.controller.ts, balance.routes.ts

## Domain entities
- **LeaveRequest** — Represents a leave application submitted by an employee. (attributes: id, employeeId, leaveType, startDate, endDate, status)
- **LeaveBalance** — Tracks the leave balances for each employee. (attributes: employeeId, annualLeave, sickLeave, emergencyLeave)

# PLAN.md — Build the leave management module

_Generated by Gestalt planning layer (feature 45fe91b3)._

## Feature description

Build the leave management module. Employees apply for annual, sick, and emergency leave. Managers approve or reject. System tracks leave balances.

## Modules
- **Leave** (`src/modules/leave`) — owns: Leave request types, Leave request persistence, Leave submission workflow, Leave approval and rejection workflow, Overlap validation orchestration
- **Balance** (`src/modules/balance`) — owns: Leave balance types, Leave balance persistence, Balance availability checks, Balance usage updates
- **Employee** (`src/modules/employee`) — owns: Employee types, Employee persistence, Employee lookup, Manager-subordinate relationship validation
- **Policy** (`src/modules/policy`) — owns: Leave policy types, Leave policy persistence, Policy lookup by leave type
- **Shared Database** (`src/shared/db`) — owns: PostgreSQL connection lifecycle
- **Shared Repository Utilities** (`src/shared/base-repository.ts`) — owns: Common database query helpers
- **Shared Middleware** (`src/shared/middleware`) — owns: Authentication middleware, Role authorization middleware, Centralized error handling
- **Shared Types** (`src/shared/types`) — owns: Shared enums, Shared request context types, Shared error classes

## Domain entities
- **Employee** — Represents the user requesting leave or acting as the assigned manager for approvals. (attributes: id, name, email, role, managerId)
- **LeaveRequest** — Represents an employee leave application and its approval lifecycle. (attributes: id, employeeId, type, startDate, endDate, status, managerId, managerComment, createdAt)
- **LeaveBalance** — Tracks available and consumed leave entitlement for each employee by leave type and year. (attributes: employeeId, leaveType, totalDays, usedDays, year)
- **LeavePolicy** — Defines configurable constraints used when validating leave requests. (attributes: id, leaveType, defaultDaysPerYear, maxConsecutiveDays, requiresApproval, createdAt)

## Feature description

Build the leave management module. Employees apply for annual, sick, and emergency leave. Managers approve or reject. System tracks leave balances.

- **leave** (`src/modules/leave`) — owns: leave.model.ts, leave.repository.ts, leave.service.ts, leave.routes.ts, leave.test.ts
- **balance** (`src/modules/balance`) — owns: balance.model.ts, balance.repository.ts, balance.service.ts
- **policy** (`src/modules/policy`) — owns: policy.model.ts, policy.repository.ts, policy.service.ts, policy.routes.ts

## Domain entities
- **LeaveRequest** — Tracks leave requests submitted by employees for various leave types. (attributes: id, employeeId, type, startDate, endDate, status, managerId, managerComment, createdAt)
- **LeaveBalance** — Maintains the leave balance for each employee based on leave type. (attributes: employeeId, leaveType, totalDays, usedDays, year)
- **LeavePolicy** — Defines the policies governing leave types and their respective rules. (attributes: id, leaveType, defaultDaysPerYear, maxConsecutiveDays, requiresApproval, createdAt)
